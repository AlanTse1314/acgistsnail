# snail（蜗牛）

#### 介绍
一款下载工具，主要支持下载类型：FTP、HTTP、BT、ED2K。

#### 技术
|技术|版本|
|:-|:-|
|JAVA|11|
|JAVAFX|11.0.2|

#### 依赖模块
jdeps --list-deps *.jar
